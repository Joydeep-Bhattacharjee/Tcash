১. বর্তমান অবস্থা বনাম আমাদের সলিউশন

বর্তমানে যা হয়: কাস্টমার অ্যাপ থেকে কমপ্লেইন করলে সেটি সরাসরি সাপোর্ট এজেন্টের ড্যাশবোর্ডে এসে জমা হয়। এজেন্টকে ম্যানুয়ালি কাস্টমারের কথা পড়তে হয়, অন্য একটি প্যানেলে গিয়ে ট্রানজ্যাকশন হিস্ট্রি মেলাতে হয়, তারপর সিদ্ধান্ত নিয়ে টাইপ করে রিপ্লাই দিতে হয়। এতে প্রচুর সময় নষ্ট হয় এবং ভুলের সম্ভাবনা থাকে।

TCash SuperCop যুক্ত হওয়ার পর: কাস্টমারের কমপ্লেইন এজেন্টের কাছে যাওয়ার আগে আমাদের API-এর ভেতর দিয়ে ফিল্টার হয়ে যাবে। এজেন্ট যখন ড্যাশবোর্ড খুলবেন, তিনি শুধু কমপ্লেইনই দেখবেন না, বরং এআই (AI) এর তৈরি করা রেডিমেড অ্যানালাইসিস এবং সেফ রিপ্লাইও দেখতে পাবেন।

২. টিক্যাশ কোথায় এবং কীভাবে কানেক্ট করবে? (Integration Flow)

টিক্যাশ মূলত তাদের Internal CRM (Customer Relationship Management) সিস্টেমের সাথে আমাদের API কানেক্ট করবে। নিচের ৪টি ধাপে এই কানেকশনটি কাজ করবে:

  • ধাপ ১: ডাটা কালেকশন (The Trigger)
    কাস্টমার যখন টিক্যাশ অ্যাপের 'সাপোর্ট' অপশনে গিয়ে লিখবে "আমি ভুলে অন্য নাম্বারে টাকা পাঠিয়েছি", তখন টিক্যাশের নিজস্ব সার্ভার সেই কমপ্লেইনটি রিসিভ করবে।
  • ধাপ ২: পেলোড তৈরি (Payload Generation)
    টিক্যাশের সার্ভার তাদের মেইন ডাটাবেস থেকে ঐ কাস্টমারের সর্বশেষ ৫টি ট্রানজ্যাকশন হিস্ট্রি টেনে আনবে। এরপর কমপ্লেইন এবং হিস্ট্রি— এই দুটোকে মিলিয়ে একটি JSON প্যাকেট তৈরি করবে।
  • ধাপ ৩: API কল (The Webhook)
    টিক্যাশের সার্ভার সেই JSON প্যাকেটটি নিয়ে আমাদের সিস্টেমে একটি POST Request পাঠাবে: https://sust01.covers.bd/analyze-ticket
  • ধাপ ৪: সাপোর্ট ড্যাশবোর্ডে প্রদর্শন (Agent View)
    আমাদের API ৩০ সেকেন্ডের মধ্যে অ্যানালাইসিস করে একটি স্ট্রাকচার্ড JSON রেসপন্স টিক্যাশের সার্ভারে ফেরত পাঠাবে। টিক্যাশের নিজস্ব সাপোর্ট ড্যাশবোর্ডে তখন নতুন কিছু ফিল্ড (AI Verdict, Routing, Priority, Draft Reply) যুক্ত হবে।

৪. টিক্যাশের লাভ কী? (Business Value)

  1. Auto-Routing: রিফান্ডের মেসেজ রিফান্ড টিমে, আর ফ্রডের মেসেজ ফ্রড টিমে অটোমেটিকভাবে চলে যাবে।
  2. Zero Safety Violations: এআই-এর সেফটি ফিল্টারের কারণে কোনো এজেন্ট ভুল করেও কাস্টমারকে "আপনার পিন দিন" বলতে পারবে না বা অথরিটি ছাড়া "টাকা ফেরত দিচ্ছি" বলতে পারবেবিধা পাবে না।
  3. Handle Massive Scale: ঈদ বা ক্যাশব্যাক ক্যাম্পেইনের সময় যখন ঘণ্টায় হাজার হাজার কমপ্লেইন আসবে, তখন এই API সাপোর্ট টিমকে কলাপ্স হওয়া থেকে বাঁচাবে।
